# AI Career Assistant Platform

An AI-powered career development platform built using Django and NLP.

## 🚀 Features

* Resume Analyzer with ATS Score
* AI Interview Question Generator
* Resume vs Job Description Match Score
* AI Career Roadmap Generator
* Job Application Tracker
* User Authentication System
* Interactive Dashboard

## 🧠 AI Capabilities

* Resume skill extraction using NLP
* ATS-style resume scoring
* AI-generated interview questions
* AI career roadmap generation
* Resume vs Job description matching

## 🛠 Tech Stack

### Backend
* Python
* Django

### AI / NLP
* spaCy
* Transformers (HuggingFace)

### Frontend
* HTML
* CSS
* Bootstrap

### Database
* SQLite
